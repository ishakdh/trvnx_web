Google Play Protect Compliance Notice:

The application "Lindux Staff Pro24" (com.trvnx.protector) is a proprietary B2B Enterprise Mobility Management (EMM) tool. It is intentionally excluded from the public Google Play Store as it requires **Device Owner provisioning** via QR code to secure organizational hardware assets against unauthorized use.

1. Purpose of the Application

Lindux Staff Pro is a specialized device security and management application built exclusively for corporate clients. Its primary function is to secure hardware assets deployed under organizational usage agreements. By establishing Device Owner privileges, the software ensures physical asset integrity throughout its operational lifecycle.

2. Required Permissions

To function as an organizational security tool, the application requires BIND_DEVICE_ADMIN (Device Owner) privileges to execute the following protocols:

  • Restrict access to the device UI in the event of a security breach.
  • Disable Safe Mode and Factory Resets to prevent unauthorized circumvention.
  • Prevent application uninstallation by unauthorized users.

3. Data Collection

We do NOT collect personal browsing history, SMS messages, user photos, or personal contacts. The application only transmits hardware telemetry (Android ID, Network Status, and Security State) to maintain management integrity.
